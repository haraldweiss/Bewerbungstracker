Key Design Decisions (iOS HIG)

  • Bottom Tab Bar: iOS standard navigation, always visible
  • Detail Cards: Full-width cards with left border for status, company name prominent
  • Color Coding: Status colors (green=interview, orange=applied, blue=offer, purple=pending)
  • Filter Buttons: Horizontal scroll, pill-shaped, tappable
  • Primary Actions: Bottom position (Create, Logout) for thumb-friendly UX
  • Typography: System fonts (San Francisco), clear hierarchy
  • Spacing: 8px, 12px, 16px grid for consistency
  • Rounded Corners: 6px cards, 3px status indicators
